New Vocabulary üìö

if statement
Code that only runs when something is True - like "if it's raining, bring umbrella"
Condition
A test that results in True or False - like checking if your phone battery > 20%
Boolean
A value that's either True or False - like a light switch (on/off)
Comparison Operator
Symbols that compare values: == (equals), > (greater), < (less), etc.
elif (else if)
Another condition to check if the first one was False

How If Statements Work üîç

if condition: # This code runs if condition is True # Notice the indentation! print("Condition was True!") # This code always runs (not indented) print("After the if statement")

Key Rules:

  • ‚úÖ End the if line with a colon :
  • ‚úÖ Indent the code that should run (use Tab or 4 spaces - just be consistent!)
  • ‚úÖ The condition must be True or False
  • ‚úÖ Code continues normally after the indented block

Common Mistake! Forgetting the colon or indentation will cause errors!

Comparison Operators ‚öñÔ∏è

Operator Meaning Example Result
== Equal to 5 == 5 True
!= Not equal to 5 != 3 True
> Greater than 10 > 5 True
< Less than 3 < 7 True
>= Greater or equal 5 >= 5 True
<= Less or equal 4 <= 8 True
Remember: Use == for comparison, = for assignment!
x = 5 assigns 5 to x
x == 5 checks if x equals 5

elif - Multiple Choices! üéØ

Use elif when you have multiple conditions to check:

grade = 85 if grade >= 90: print("A - Excellent! üåü") elif grade >= 80: print("B - Great job! üëç") elif grade >= 70: print("C - Good work! ‚úì") elif grade >= 60: print("D - Keep trying!") else: print("F - Need more practice")

How it works:

‚Ä¢ Checks conditions from top to bottom

‚Ä¢ Stops at the first True condition

‚Ä¢ Only one block runs!

Logical Operators - Combine Conditions! üîó

and - Both must be True

age = 16 has_permit = True if age >= 16 and has_permit: print("You can practice driving! üöó")

or - At least one must be True

day = "Saturday" if day == "Saturday" or day == "Sunday": print("It's the weekend! üéâ")

not - Flips True/False

raining = False if not raining: print("No rain - let's go outside! ‚òÄÔ∏è")

Remember: and needs BOTH, or needs ONE, not flips the value!

Common Mistakes to Avoid! ‚ö†Ô∏è

‚ùå Using = instead of ==

# WRONG if score = 100: # This is assignment! # CORRECT if score == 100: # This is comparison!

‚ùå Forgetting the colon

# WRONG if age > 18 # Missing colon! # CORRECT if age > 18: # Don't forget the colon!

‚ùå Wrong indentation

# WRONG if score > 90: print("Great!") # Not indented! # CORRECT if score > 90: print("Great!") # Indented properly!
Pro Tip: Python uses indentation to know what code belongs to the if statement!
