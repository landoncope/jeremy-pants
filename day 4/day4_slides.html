Another Vocabulary Word üìö

Git
A version control system that tracks changes in your code. Think of it like a super-powered "undo" button that remembers every change you've ever made to your files!

Key Git Concepts:

  • üì∏ Commit = A snapshot of your code at a specific point
  • üå≥ Branch = A separate version to try new ideas
  • üîÑ Push/Pull = Send/receive changes to/from GitHub
  • üç¥ Fork = Make your own copy of someone's project

üìù Git is different from GitHub! Git is the tool, GitHub is a website that uses Git.

Install Git üîß

Git = The tool that tracks changes to your code

For Windows:

  • Download from git-scm.com
  • Run the installer (use default settings)
  • Open "Git Bash" when done

For Mac:

  • Open Terminal (find it in Applications ‚Üí Utilities)
  • Type: git --version
  • If not installed, it will prompt you to install
