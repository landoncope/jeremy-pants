Today's New Functions! üÜï

VLOOKUP

Vertical Lookup - Like a phone book! Look up a value in one column, get info from another column in the same row.

Think: "Find this person's name, tell me their phone number"

IF Statement

Makes decisions! If something is true, do one thing. If false, do another.

Like: "IF it's raining, bring umbrella, ELSE bring sunglasses"

Real World: VLOOKUP is used in every business to match data. IF statements automate decision-making!

Task 2: Field Trip Fee Formula üí∞

The Challenge:

Everyone pays the same field trip fee (shown in cell M2: $15.00)

In cell F3, type: =$M$2

Why the dollar signs?

$M$2 is an absolute reference - it won't change when we copy the formula down!

After typing the formula:

  1. Press Enter
  2. Click cell F3 again
  3. Copy (Ctrl+C or Cmd+C)
  4. Select F4 through the last student
  5. Paste (Ctrl+V or Cmd+V)

Check: All students should show $15.00!

Task 3: VLOOKUP for Shirt Prices üëï

How VLOOKUP Works:

=VLOOKUP(what_to_find, where_to_look, which_column, exact_match)

Our Lookup Table (Data sheet)

SizePrice
S$8.00
M$9.00
L$10.00
XL$12.00

Think of it like: "Look up the size in column E, find it in the Data sheet, tell me the price from column 2"

Writing the VLOOKUP Formula üîç

In cell G3, type exactly:

=VLOOKUP(E3,Data!A:B,2,FALSE)

Breaking it down:

  • E3 = The shirt size to look up
  • Data!A:B = Look in columns A and B of the Data sheet
  • 2 = Return the value from the 2nd column (Price)
  • FALSE = Find an exact match only

After entering: Copy this formula down to all students. Each should show the correct price for their size!

Common Mistake: Forgetting "Data!" before A:B. This tells Sheets to look in the Data sheet!

Task 4: Calculate Total Due üíµ

Simple Addition!

Each student's total = Field Trip Fee + Shirt Price

In cell H3, type: =F3+G3

Copy down for all students!

Example results:

  • Small shirt: $15 + $8 = $23.00
  • Medium shirt: $15 + $9 = $24.00
  • Large shirt: $15 + $10 = $25.00
  • XL shirt: $15 + $12 = $27.00

Task 5: IF Statement for Actions üìß

The Logic:

IF the form is not received ‚Üí Email Parents
ELSE ‚Üí OK

In cell J3, type:

=IF(I3="No","Email Parents","OK")

How it reads:

  • IF cell I3 equals "No"
  • THEN show "Email Parents"
  • ELSE show "OK"

Copy down! You should see "Email Parents" for students with "No" in Form Received, and "OK" for those with "Yes"

Real World: Schools use IF statements to automatically flag missing permissions, unpaid fees, and incomplete registrations!

Common Problems & Solutions üîß

VLOOKUP shows #N/A?

  • Check spelling of shirt sizes (must match exactly!)
  • Make sure you typed Data!A:B not just A:B
  • FALSE must be spelled correctly

Formulas not copying down?

  • Select the cell with the formula
  • Copy (Ctrl+C)
  • Select ALL cells where you want it
  • Paste (Ctrl+V)

Summary functions showing 0 or error?

  • Check your range (A3:A50 etc.)
  • Make sure you're counting the right column
