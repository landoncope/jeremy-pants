üìñ Vocabulary Terms

‚ñº Day 1: Introduction & Binary Basics
Bit
The smallest unit of data in computing, representing a single 0 or 1.
Byte
A unit of digital information that consists of 8 bits.
Repository
A storage location for code and project files, often used for collaboration and version control (e.g., on GitHub).
‚ñº Day 2: Binary Math & Computer Basics
Operating System
The main software that manages a computer's hardware and software resources, and provides common services for computer programs. Examples include Windows, MacOS, Linux, Android, and iOS.
File Extension
The suffix at the end of a filename that indicates what type of file it is (for example, .docx for Word documents, .jpg for images, .exe for executable programs).
Hard Drive
A data storage device used for storing and retrieving digital information. It retains data even when the computer is turned off and is used to store files, programs, and the operating system.
RAM (Random Access Memory)
A type of computer memory that is used for temporary storage while a computer is running. It allows data to be read and written quickly, but all information is lost when the power is turned off.
‚ñº Day 3: Command Line & Web Basics
Gigabyte (GB)
A unit of digital storage equal to approximately 1 billion bytes (technically 1,024 megabytes). Think of it like a large box that can hold about 250 songs or a 30-minute HD video.
Command Line (Terminal)
A text-based way to control your computer by typing commands instead of clicking with a mouse. On Mac, it's called Terminal. On Windows, it's called Command Prompt or PowerShell.
HTML
HyperText Markup Language - The standard language for creating web pages. It uses tags to tell the browser what each part of the page is: headings, paragraphs, images, links, etc.
‚ñº Day 4: HTML Basics
Open Source
Software with source code that anyone can inspect, modify, and share. It's like a recipe that everyone can see, improve, and use for free! Examples include Linux, Firefox, Android, and VS Code.
‚ñº Day 5: Git & GitHub
Git
A version control system that tracks changes to your files over time. Think of it like Google Docs' version history, but for code! It lets you save snapshots of your work and collaborate with others.
Tag (HTML)
A piece of code that tells the browser how to display content. Tags usually come in pairs with an opening tag like <p> and a closing tag like </p>. They're like instructions that wrap around your content.
‚ñº Day 6: Spreadsheets
Cell
A single box in a spreadsheet where data lives. Like a parking spot for one piece of information. Named by column letter and row number (A1, B2, etc.)
Formula
A calculation that starts with = sign. Like giving the spreadsheet math homework that it solves instantly. Example: =A1+B1
Function
A pre-built formula that does common tasks. Like shortcuts on your phone - instead of typing a whole address, you just tap "Home". Example: =SUM(A1:A10)
Range
A group of cells selected together. Like highlighting multiple messages to delete at once. Written as A1:B5 (from top-left to bottom-right)
Cell Reference
Using a cell's address in a formula. Like tagging a friend in a post - you're pointing to their content. When you type =A1*2, you're referencing cell A1
‚ñº Day 7: More Spreadsheets
VLOOKUP
Vertical Lookup - searches for a value in the first column of a table and returns a value in the same row from another column. Like a phone book: find the name, get the number!
IF Statement
A function that makes decisions. Tests if something is true or false, then does different things based on the result. Format: =IF(test, do_if_true, do_if_false)
ROUND Function
Rounds a number to a specified number of digits. =ROUND(3.14159, 2) gives you 3.14. Perfect for making messy numbers clean!
‚ñº Day 8: Spreadsheet Project
Absolute Reference
A cell reference that doesn't change when copied. Uses $ signs like $A$1. Think of it as "locking" a cell address so it always points to the same spot, like a GPS coordinate!
Relative Reference
A cell reference that changes when copied. Like A1 (no $ signs). It adjusts based on where you paste it, like giving directions from "your current location"
Fill Down
Copying a formula to multiple cells below. Select the cell with your formula, then drag down or copy-paste to apply the same formula to a whole column. Super time-saver!
‚ñº Day 9: Python Basics
print()
Makes the computer show text on the screen - like posting a message!
Variable
A container that stores information - like a labeled box for your stuff
String
Text in quotes - like a text message you send to friends
Integer
A whole number with no decimals - like your age or score in a game
Float
A number with decimals - like your GPA, height, or the price of something
Comment
Notes in your code that the computer ignores - like writing notes in the margins of a textbook
‚ñº Day 10: Python Decisions & Logic
if statement
Code that only runs when something is True - like "if it's raining, bring umbrella"
Condition
A test that results in True or False - like checking if your phone battery > 20%
Boolean
A value that's either True or False - like a light switch (on/off)
Comparison Operator
Symbols that compare values: == (equals), > (greater), < (less), etc.
elif (else if)
Another condition to check if the first one was False
Logical Operator
Combines conditions using and, or, not - like "if hungry AND have money, buy food"
‚ñº Day 11: Python Loops, Lists & Functions
Loop
Code that repeats automatically - like playing a song on repeat
List
A container that holds multiple values - like a playlist of songs
Function
Reusable code with a name - like a recipe you can use over and over
Iteration
One cycle through a loop - like one lap around a track
Index
Position number in a list (starts at 0) - like seat numbers in a theater
Return
When a function sends back a value - like a vending machine giving you change
‚ñº Day 14: Building Your First Website with AI
4K Resolution
A display resolution of approximately 4,000 pixels wide (3840 x 2160 pixels), providing four times the detail of regular HD. Think of it like looking through no window at all - everything is super crisp!
E-commerce
Electronic Commerce - Buying and selling goods or services over the internet. Every time you buy something on Amazon or sell clothes on Depop, you're doing e-commerce!
Modem
A device that connects your house to the internet by talking to your Internet Service Provider (ISP). It's like a water pipe bringing water to your house.
Router
A device that takes the internet connection from your modem and shares it with all your devices by creating a WiFi network. It's like a sprinkler system spreading water around your yard.
Static Website
A website where every visitor sees the exact same content. The pages are pre-built and don't change unless you edit the code. Like a printed poster - everyone sees the same thing.
GitHub Codespaces
A cloud-based development environment that runs VS Code in your browser. It's like having a complete coding setup that works anywhere without installation!
‚ñº Day 16: Building Games with AI
Bandwidth
The maximum amount of data that can be transmitted over an internet connection in a given amount of time, usually measured in Mbps (megabits per second). Think of it like a pipe carrying water - a bigger pipe (more bandwidth) can carry more water (data) at once. High bandwidth = fast downloads and smooth streaming. Low bandwidth = buffering and slow loads.
Cache
A temporary storage location where your computer, phone, or browser saves copies of files, images, and data to make things load faster the next time you need them. Like keeping your favorite snacks in your backpack instead of going to the kitchen every time - your browser keeps website copies so it doesn't have to download them again!
‚ñº Day 17: Mastering AI Prompts
Streaming
A method of transmitting or receiving data (especially video and audio) over a network as a continuous flow, allowing playback to begin while the rest is still being received. Like watching a YouTube video that starts playing right away - you don't wait for the entire video to download first. The data "streams" to you in real-time!
Algorithm
A step-by-step set of instructions or rules designed to solve a specific problem or perform a specific task. Think of a recipe - it's step-by-step instructions to make a cake. In computing, algorithms power everything from your social media feed (TikTok/Instagram algorithms) to Google search results to Netflix recommendations.
